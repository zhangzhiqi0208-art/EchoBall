🎯 Prompt对比

🔴 旧Prompt格式

🟢 新Prompt格式

{ "sentiment_type": "positive/negative/neutral", "confidence_level": "high/medium/low", "reasons": ["原因1", "原因2", "原因3"], "intensity": "mild/moderate/strong" }

特点：

  • 英文标准分类
  • 置信度评估
  • 多维度分析依据
  • 更精确的强度分级

📊 新Prompt优势

🎯 更精确的分类

  • sentiment_type：使用国际标准positive/negative/neutral
  • confidence_level：评估分析的可信度
  • reasons：提供具体的分析依据
  • intensity：更细致的强度分级

🔍 更详细的分析

  • 多维度评估：情感类型+置信度+依据+强度
  • 结构化输出：便于程序解析和处理
  • 可追溯性：提供分析依据列表
  • 标准化：使用国际通用的情感分析标准
